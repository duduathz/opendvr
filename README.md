# opendvr
open DVR/NVR
一个开源DVR/NVR项目，主要应用于安防视频监控行业。

# 目标
探讨使用C、Go、Lua等语言混合编程并应用于嵌入式行业，以期待提高嵌入式开发效率。

# 使用语言特性
* C：操作硬件、高效内存使用
* Go：开发效率与性能平衡
* Lua：灵活胶水语言

# 规避语言劣势
* C：各种非法指针、段错误、字符串操作等
